h1. Pomoduino

Por grupo Mexicali Open Source (mxlOS)!

Pomoduino es un proyecto para crear un Pomodoro timer con Arduino.

h2. Dependencias de compilacion

* Actualizar Arduino IDE a al menos 1.0
* "Obtener":https://github.com/mxlOS/pomoduino-libraries las librerias de LED, FSM y Button y agregarlas a tu directorio _sketchbook/libraries_

h2. Filosofia del proyecto

* Todo mundo esta invitado a colaborar
* Se aceptan comentarios y sugerencias para mejorar (somos nuevos en Arduino!)
* No hay prisa
